## 🏔️ About Flux
Flux is a minimalist, high-performance task management tool designed to reduce cognitive load. While other tools drown you in features, Flux focuses on speed, aesthetics, and a "low-energy" UI that lets your work take center stage.

Built for developers and creators who appreciate the "Linear" feel but want something faster and more personal.

## ✨ Key Features (v0.1 Beta)
* **Calm Dark UI:** A refined dark theme with high-density information and zero visual noise.
* **Context-Aware Side Panel:** Manage task details without losing sight of your board or list.
* **Snappy Interactions:** Hand-tuned animations for a fast, responsive desktop experience.
* **Flexible Views:** Switch between a focused List view and a strategic Kanban Board.
* **Command-First Workflow:** Draggable modals and inline task creation to keep your hands on the keyboard.
* **Safety Net:** Instant task deletion with a seamless "Undo" mechanism.

## 🛠️ Tech Stack
* **Core:** Rust (Tauri) for a lightweight desktop shell.
* **Frontend:** TypeScript + React with a focus on component modularity.
* **Styling:** Tailored CSS for a premium dark-mode experience.
